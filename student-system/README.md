&#x20;🌟 项目简介

这是一个基于Win32 API 和 C++ STL开发的桌面级管理系统。本项目专门针对Visual Studio 2010 进行了兼容性优化，移除了现代 C++ 特性以确保在老版本编译器上完美运行。

!\[学生管理系统](stu.png)

✨ 核心优化与特性

\-🎨 紫色 UI: 通过拦截 `WM\_CTLCOLORSTATIC` 和 `WM\_CTLCOLORDLG` 消息，自定义了淡紫色（RGB 245, 245, 255）的视觉风格。

\- 💾 数据持久化: 引入二进制文件流（fstream），数据自动保存至 `students.dat`，实现“关机不丢失”。

\- ⚡ 算法适配: 针对 VS2010 编译器，使用传统的迭代器（Iterator）和函数指针（Function Pointer）实现高效的增删改查逻辑。

\- 📊 格式化对齐: 列表框采用等宽格式输出，确保多条数据排列整齐。



\## 🛠️ 编译与运行指南

1.环境: Windows + Visual Studio 2010。

2\. 配置: 

&#x20;  - 项目属性 -> 配置属性 -> 常规 -> 字符集改为使用多字节字符集。

3\. 数据文件: 程序运行后会自动在根目录生成 `students.dat`。
